# 🧠 Predicting Obesity Levels Using Machine Learning

## 👨‍💻 Author

**Lad Kevinkumar Chandubhai**

---

## 📌 Project Overview

This project uses supervised machine learning techniques to analyze how lifestyle, dietary habits, and demographic factors affect obesity levels. The goal is to predict obesity categories and identify the most important contributing factors.

---

## 📊 Dataset

* **Name:** Obesity Levels Dataset
* **Source:** Kaggle
* **Link:** https://www.kaggle.com/code/ladkevinkumar/obesity-level-check
* **Rows:** 2,111
* **Columns:** 17
* **Target Variable:** `NObeyesdad`

---

## 🎯 Objectives

* Analyze factors influencing obesity
* Answer research questions using data analysis
* Build classification models
* Compare model performance

---

## ❓ Research Questions

1. How do dietary habits influence obesity?
2. What is the relationship between physical activity and screen time?
3. How do age and gender affect obesity levels?
4. How do smoking and alcohol consumption relate to obesity?
5. Which features are most important for prediction?
6. How does water intake affect obesity?
7. Which machine learning model performs best?

---

## ⚙️ Methodology

### Data Processing

* Data cleaning
* Encoding categorical variables
* Train-test split

### Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* KNN
* XGBoost

### Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score

## 🧪 Results

* Lifestyle and diet significantly affect obesity
* Physical activity and screen time are key factors
* Water intake has a negative relationship with obesity
* Random Forest / XGBoost give the best performance
